ข้อกำหนดการใช้งาน

วันที่มีผลบังคับใช้:

ยินดีต้อนรับ! ข้อกำหนดนี้ควบคุมการใช้งาน Affilitics.me ที่เชื่อมต่อกับ Facebook API

1. การยอมรับข้อกำหนด

การใช้งานแอปพลิเคชันนี้ถือว่าคุณยอมรับและตกลงปฏิบัติตามข้อกำหนดทั้งหมด หากไม่ยอมรับ กรุณาหยุดใช้งานทันที

2. คำอธิบายบริการ

Affilitics.me เป็นเครื่องมือสำหรับ:

  • วิเคราะห์ประสิทธิภาพแคมเปญโฆษณา Facebook
  • ติดตามสถิติและ ROI ของการลงทุนโฆษณา
  • สร้างรายงานและกราฟแสดงผลการดำเนินงาน
  • ส่งออกข้อมูลเพื่อการวิเคราะห์เพิ่มเติม

3. การใช้งานที่อนุญาต

คุณสามารถใช้แอปพลิเคชันนี้เพื่อ:

  • วิเคราะห์ข้อมูลโฆษณาของคุณเอง
  • สร้างรายงานสำหรับธุรกิจของคุณ
  • ติดตามประสิทธิภาพการลงทุนโฆษณา
  • ปรับปรุงกลยุทธ์การตลาด

4. การใช้งานที่ห้าม

ห้าม:

  • ใช้งานเพื่อวัตถุประสงค์ที่ผิดกฎหมาย
  • เข้าถึงข้อมูลของผู้อื่นโดยไม่ได้รับอนุญาต
  • คัดลอก, แจกจ่าย, หรือดัดแปลงแอปพลิเคชัน
  • ใช้งานในลักษณะที่อาจทำให้เกิดความเสียหาย
  • ละเมิดข้อกำหนดของ Facebook

5. บัญชีผู้ใช้และความปลอดภัย

คุณมีหน้าที่:

  • รักษาข้อมูลการเข้าสู่ระบบให้ปลอดภัย
  • แจ้งเราทันทีหากพบการใช้งานที่ผิดปกติ
  • ใช้งานด้วยความระมัดระวังและรับผิดชอบ
  • ปฏิบัติตามนโยบายของ Facebook

6. ความเป็นส่วนตัวและข้อมูล

การใช้งานแอปพลิเคชันนี้อยู่ภายใต้นโยบายความเป็นส่วนตัวของเรา ซึ่งอธิบายวิธีการเก็บรวบรวมและใช้ข้อมูลของคุณ

7. การเชื่อมต่อ Facebook API

เมื่อเชื่อมต่อกับ Facebook:

  • คุณให้สิทธิ์เราในการเข้าถึงข้อมูลโฆษณาของคุณ
  • เราจะเข้าถึงเฉพาะข้อมูลที่จำเป็นสำหรับการให้บริการ
  • คุณสามารถยกเลิกการเชื่อมต่อได้ตลอดเวลา
  • การใช้งานต้องปฏิบัติตามข้อกำหนดของ Facebook

8. ข้อจำกัดความรับผิดชอบ

เราไม่รับผิดชอบต่อ:

  • ความเสียหายที่เกิดจากการใช้งานแอปพลิเคชัน
  • ความไม่ถูกต้องของข้อมูลจาก Facebook API
  • การหยุดชะงักของบริการ Facebook
  • การสูญหายของข้อมูลหรือกำไร

9. การยกเลิกบริการ

เราสงวนสิทธิ์ในการ:

  • ยกเลิกหรือระงับการให้บริการโดยไม่แจ้งล่วงหน้า
  • ปรับปรุงหรือเปลี่ยนแปลงฟีเจอร์
  • บล็อกการเข้าถึงหากมีการใช้งานที่ไม่เหมาะสม

10. การเปลี่ยนแปลงข้อกำหนด

เราอาจปรับปรุงข้อกำหนดนี้เป็นครั้งคราว การเปลี่ยนแปลงจะมีผลทันทีเมื่อประกาศบนแอปพลิเคชัน

11. กฎหมายที่ใช้บังคับ

ข้อกำหนดนี้อยู่ภายใต้กฎหมายไทย ข้อพิพาทจะได้รับการแก้ไขโดยศาลไทย

12. การติดต่อ

หากมีคำถามเกี่ยวกับข้อกำหนดนี้ กรุณาติดต่อ:

  • อีเมล: support@yourcompany.com
