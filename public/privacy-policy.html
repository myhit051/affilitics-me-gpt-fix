นโยบายความเป็นส่วนตัว

วันที่มีผลบังคับใช้:

สำคัญ: นโยบายนี้อธิบายวิธีที่เราเก็บรวบรวม ใช้ และปกป้องข้อมูลของคุณเมื่อใช้งาน Affilitics.me ที่เชื่อมต่อกับ Facebook API

1. ข้อมูลที่เราเก็บรวบรวม

เมื่อคุณเชื่อมต่อบัญชี Facebook กับแอปพลิเคชันของเรา เราจะเก็บรวบรวมข้อมูลดังต่อไปนี้:

  • ข้อมูลโปรไฟล์พื้นฐาน: ชื่อ, อีเมล, รูปโปรไฟล์
  • ข้อมูลบัญชีโฆษณา: รายชื่อบัญชีโฆษณา Facebook ที่คุณมีสิทธิ์เข้าถึง
  • ข้อมูลแคมเปญ: ชื่อแคมเปญ, สถิติการทำงาน, งงบประมาณ
  • ข้อมูลสถิติ: จำนวนการแสดงผล, คลิก, ค่าใช้จ่าย, อัตราการแปลง

2. วิธีการใช้ข้อมูล

เราใช้ข้อมูลที่เก็บรวบรวมเพื่อ:

  • แสดงสถิติและรายงานการทำงานของแคมเปญโฆษณา
  • วิเคราะห์ประสิทธิภาพการลงทุนโฆษณา (ROI)
  • สร้างกราฟและแผนภูมิเพื่อการวิเคราะห์
  • ส่งออกข้อมูลในรูปแบบ Excel หรือ CSV
  • ปรับปรุงและพัฒนาฟีเจอร์ของแอปพลิเคชัน

3. การแบ่งปันข้อมูล

เราไม่แบ่งปัน ขาย หรือเปิดเผยข้อมูลส่วนบุคคลของคุณให้กับบุคคลที่สาม ยกเว้น:

  • เมื่อได้รับความยินยอมจากคุณอย่างชัดเจน
  • เมื่อกฎหมายกำหนดให้ต้องเปิดเผย
  • เพื่อปกป้องสิทธิ์และความปลอดภัยของเราและผู้ใช้อื่น

4. การรักษาความปลอดภัยข้อมูล

เราใช้มาตรการรักษาความปลอดภัยที่เหมาะสม เช่น:

  • การเข้ารหัสข้อมูลระหว่างการส่งผ่าน (HTTPS/TLS)
  • การเข้ารหัสข้อมูลที่เก็บไว้ในเครื่อง
  • การจำกัดการเข้าถึงข้อมูลเฉพาะผู้ที่จำเป็น
  • การตรวจสอบและอัปเดตระบบรักษาความปลอดภัยอย่างสม่ำเสมอ

5. การเก็บรักษาข้อมูล

เราจะเก็บข้อมูลของคุณไว้เท่าที่จำเป็นสำหรับการให้บริการ หรือตามที่กฎหมายกำหนด คุณสามารถขอให้ลบข้อมูลได้ตลอดเวลา

6. สิทธิ์ของคุณ

คุณมีสิทธิ์ในการ:

  • เข้าถึงและตรวจสอบข้อมูลส่วนบุคคลของคุณ
  • แก้ไขข้อมูลที่ไม่ถูกต้อง
  • ขอให้ลบข้อมูลของคุณ
  • ถอนความยินยอมในการใช้ข้อมูล
  • ส่งออกข้อมูลของคุณ

7. การใช้ Facebook API

แอปพลิเคชันนี้ใช้ Facebook API เพื่อ:

  • เข้าถึงข้อมูลบัญชีโฆษณาที่คุณให้สิทธิ์
  • ดึงข้อมูลสถิติแคมเปญและโฆษณา
  • อ่านข้อมูลประสิทธิภาพการโฆษณา

เราปฏิบัติตามนโยบายและข้อกำหนดของ Facebook อย่างเคร่งครัด

8. การเปลี่ยนแปลงนโยบาย

เราอาจปรับปรุงนโยบายนี้เป็นครั้งคราว การเปลี่ยนแปลงที่สำคัญจะแจ้งให้คุณทราบล่วงหน้า

9. การติดต่อเรา

หากคุณมีคำถามเกี่ยวกับนโยบายความเป็นส่วนตัวนี้ กรุณาติดต่อเราที่:

อีเมล: privacy@yourcompany.com

ที่อยู่: [ใส่ที่อยู่บริษัทของคุณ]

โทรศัพท์: [ใส่เบอร์โทรศัพท์ของคุณ]

หมายเหตุ: นโยบายนี้ใช้สำหรับการพัฒนาและทดสอบแอปพลิเคชัน กรุณาปรับแต่งให้เหมาะสมกับธุรกิจของคุณก่อนใช้งานจริง
